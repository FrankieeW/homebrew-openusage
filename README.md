# Homebrew Tap for OpenUsage.cc

```fish
brew install --cask frankieew/openusage/openusage-cc
```

## Publishing

This tap follows the [OpenUsage Collection](https://github.com/FrankieeW/openusage-collection#publishing) publishing flow.

1. Fork [openusage-collection](https://github.com/FrankieeW/openusage-collection)
2. Add your plugin under `plugins/<id>/`
3. `plugin.json`: `schemaVersion: 1`, `id` matches directory name
4. Open a PR
